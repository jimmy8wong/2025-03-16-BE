# Vehicles API

## Overview
API to fetch and update technical data about vehicles.

## Installation
### Prerequisites
- PHP (version 8.2)
- Composer (version 2)

### Dependencies
The application is based on `symfony/skeleton@7.2` and requires the following dependencies:
- `symfony/orm-pack`
- `symfony/maker-bundle`
- `symfony/validator`
- `orm-fixtures`
- `symfony/test-pack`
- `symfony/serializer-pack`
- `symfony/process`
